A geospatial-enabled lost and found cats tracking platform built with Next.js, Express, and Leaflet.

## Tech Stack

**Frontend:**
- Next.js 13+ with App Router
- React 18
- TypeScript
- TailwindCSS
- Leaflet Maps
- React Query
- Zustand

**Backend:**
- Node.js/Express
- TypeScript
- Geospatial database

## Quick Start

```bash
# Frontend
cd frontend
npm install
npm run dev   # Runs on :3001

# Backend
cd backend
npm install
npm run dev   # Runs on :3000
```
